## Getting Started

1. **Clone the repository**: `git clone ...`
2. **Setup Environment**: Run `./devpods/boot_linux.sh` (or appropriate script for your OS).
3. **Connect**: `tmux attach -t workspace`
4. **Start Coding**: Use `cf-swarm` or `cf-hive` to start working with agents.
